# E-commerce Reviews Data Analysis

This project analyzes an e-commerce dataset containing user ratings, reviews, and product information.

## Project Tasks
- Data cleaning and missing value removal to ensure dataset quality.
- Descriptive statistics analysis to understand users, products, reviews, and ratings.
- Data visualization using boxplots to explore relationships between ratings, user attributes, and product categories.
- Outlier detection and removal to improve data reliability.
- Correlation analysis to identify factors influencing user ratings.
- Predictive modeling with linear regression and evaluation using MSE/RMSE.
- Insights generation to highlight trends in user behavior and product review patterns.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset Features
- userId
- gender
- rating
- review
- category
- helpfulness
- item_price
- timestamp
- user_city
