# E-commerce Reviews Data Analysis

This project analyzes an e-commerce dataset containing user ratings, reviews, and product information.

## Project Tasks
- Data cleaning and missing value removal
- Descriptive statistics analysis
- Data visualization using boxplots
- Outlier detection and removal

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset Features
- userId
- gender
- rating
- review
- category
- helpfulness
- item_price
- timestamp
- user_city
